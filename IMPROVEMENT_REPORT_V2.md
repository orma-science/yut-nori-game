
# 윷놀이 게임 2차 개선 보고서: 이벤트 & 턴 스킵 시스템

## 1. 개요
요청하신 5가지 새로운 이벤트와 '한 판 쉬기(턴 스킵)' 기능을 구현 완료했습니다.
이로써 게임의 변수가 더욱 다양해지고, 전략적인 요소가 강화되었습니다.

## 2. 신규 이벤트 구현 상세
| 이벤트 명 | 설명 | 구현 내용 |
| :--- | :--- | :--- |
| **🔙 전략적 후퇴** | 2칸 뒤로 가기 | `calculateTargetNode`로 -2칸 계산하여 이동 실행 |
| **🚀 급행 열차** | 3칸 앞으로 가기 | `calculateTargetNode`로 +3칸 계산하여 이동 실행 |
| **✨ 황금 윷가락** | 한 번 더 던지기 | 별도 이동 없이 역사에 기록만 남기고, 턴이 넘어가지 않음 |
| **😱 강제 귀가 조치** | 말 잡혀가기 | 해당 말을 `pieces` 배열에서 제거하고, 대기 말(`piecesAtHome`) 개수 복구 |
| **💤 강제 동면** | 다음 턴 쉬기 | `skipNextTurnTeamIds` 상태에 팀 ID를 추가하여, 다음 차례가 왔을 때 자동으로 건너뜀 |

## 3. 핵심 로직: 턴 스킵 (Turn Skipping)
- **상태 관리**: `GameState`에 `skipNextTurnTeamIds: TeamId[]` 필드를 추가하여, 쉬어야 할 팀들을 관리합니다.
- **자동 건너뛰기**: 턴을 넘기는 시점(`shouldSwitch`가 `true`일 때)에, 다음 팀이 `skipNextTurnTeamIds` 목록에 있는지 확인합니다.
  - 목록에 있다면: "동면 중이라 턴을 건너뜁니다" 메시지를 출력하고, 스킵 목록에서 제거한 뒤, **그 다음 팀으로 턴을 한 번 더 넘깁니다.**
  - `while` 루프를 사용하여 연속으로 여러 팀이 쉬어야 하는 경우도 처리할 수 있습니다.

## 4. 기타 개선
- **이벤트 이름과 설명**: 요청하신 내용을 바탕으로 조금 더 게임의 분위기에 맞게 맛깔나는 이름으로 다듬었습니다. (예: "앗! 한 판 쉬기" -> "💤 강제 동면")
- **잡기/업기 연동**: 이벤트로 이동할 때도 기존 `executeMove` 함수를 활용하므로, 말을 잡거나 업는 규칙이 똑같이 적용됩니다.

이제 다시 `npm run dev`를 통해 업그레이드된 게임을 즐겨보세요!
